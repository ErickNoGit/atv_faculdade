# Projeto de Luminárias

Este é o projeto da atividade 1 da faculdade, onde desenvolvi um site básico utilizando HTML e CSS para apresentar informações sobre luminárias.

## Descrição

O objetivo deste projeto foi criar um site simples que demonstrasse meu conhecimento em HTML e CSS, além de fornecer informações sobre diferentes tipos de luminárias, suas características e usos.

## Funcionalidades

- Página inicial com uma breve introdução sobre o projeto e uma galeria de imagens de luminárias.
- Páginas individuais para cada tipo de luminária, contendo informações detalhadas, como características técnicas, materiais utilizados e exemplos de aplicações.
- Estilo de design limpo e moderno, com uso adequado de cores e tipografia para uma melhor experiência do usuário.

## Tecnologias Utilizadas

- HTML5
- CSS3

## Como Visualizar

1. Faça o clone deste repositório em sua máquina local.
2. Abra o arquivo `index.html` em seu navegador da web preferido.
